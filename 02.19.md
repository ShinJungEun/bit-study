< 리눅스 디렉토리 구조 >
/ : 루트
/bin : 시스템 관리자 or 일반사용자가 모두 사용가능한 명령어 저장소
/sbin : 시스템 관리자가 사용할 수 있는 명령어 저장소. 시스템 수정, 복구 관련 명령어.
 - 일반 사용자에게는 실행 권한을 제한
/etc : 시스템, 프로그램들의 환경 설정 파일들이 위치. 백업이 중요
/lib : 프로그램이 실행될 때 필요한 공유 라이브러리가 저장되어 있음
/mnt : 로그파일들. 마운트를 위한 임시 디렉토리가 위치함. 이동 디스크(CD,USB)를 마운트할 때 사용
/log : 로그파일
 - tail -f : 로그 확인 명령어 ex) tail -f catalina.out : tomcat log확인
 - less dmesg ...

usr : 설치파일
lost+found : 잃어버린 섹터. 임시 저장소


buffer : io 관련
cache

* ps : 현재 실행되고 있는 프로세스 목록을 보여줌
  ps -ef

* top : 프로세스 cpu, memory사용량 등의 전반적인 상황을 실시간 모니터링
* nice : 프로세스 우선순위를 결정
  : 숫자가 작을수록 우선순위가 높음
 - renice : 프로세스 우선순위를 변경 ex) renice 10 1611 => PID가 1611인 프로세스의 nice 값을 10으로 변경

* Model1 : JSP(View/Controller)
*Model2(MVC) : JSP(View), Servlet(Controller)

* MVC : 아키텍쳐 패턴 중하나, GUI기반 application 설계 패턴

* Model : application 정보를 처리하는 모듈(객체)
 - View : text, checkbox, 입력창 등의 사용자 인터페이스 요소
 - Controller : 클라이언트 요청, Model과 View 사이의 상호동작을 관리

* html에서는 redirect가 중요(한페이지에 머물러있지 않도록)
